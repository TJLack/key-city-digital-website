# Conversion Domination System

## 1) CTA rewrite examples
| Old CTA | New CTA (Outcome-driven) |
|---|---|
| Run Free Scan | See How Many Jobs You're Losing |
| Run Free Website Scan | Run Free 60-Second Scan |
| Book Consultation | Book Your Strategy Call |
| Submit | Get My Growth Plan & Next Steps |

## 2) CTA placement strategy (decision moments)
Every high-intent page should place both CTAs at:
1. Above the fold (hero)
2. Mid-page after value/solution section
3. After problem/agitation section
4. Final section before footer

Required dual CTAs:
- `https://WebsiteLeakDetector.com`
- `https://api.leadconnectorhq.com/widget/bookings/digital-marketing-consultation-apykl`

## 3) WebsiteLeakDetector positioning improvements
Positioning used across pages:
- Proprietary diagnostic (not generic audit)
- Core promise: "Find out how many jobs your website is losing"
- Benefit bullets: conversion leaks, trust gaps, SEO blockers
- Urgency: diagnose before spending more on ads

## 4) Booking section improvements
Booking sections should explain:
- who the call is for (Texas contractors wanting more calls/booked jobs)
- what happens next (90-day plan + priorities by impact)
- trust statements (clear timeline, budget ranges, practical steps)
- speed-to-lead expectation (rapid follow-up)

## 5) Page flow recommendations
Use this sequence on core pages:
1. Hook (problem)
2. Agitate (what is being lost right now)
3. Solution (service/system)
4. Proof (results/trust/specialization)
5. CTA (scan + booking)
6. Objection handling (cost, timeline, skepticism)

## 6) Conversion optimization checklist
- [ ] Both CTAs present on every page
- [ ] CTA labels are benefit-driven, not generic
- [ ] Booking CTA links directly to calendar
- [ ] At least 3 CTA placements on long pages
- [ ] Trust statement near booking section
- [ ] Objection-handling FAQ section present
- [ ] Clear speed-to-lead expectation on contact flow
- [ ] Monthly A/B tests for CTA wording and section order

## Testing system (what to test)
- Headline framing: pain-led vs outcome-led
- CTA text: "See How Many Jobs You're Losing" vs "Run Free 60-Second Scan"
- CTA emphasis: scan-first vs booking-first
- Mid-page CTA position: after problem vs after solution
- Booking trust copy variations
